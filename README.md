# tbaep
Pacote da Transparência Brasil para o projeto Achados e Pedidos

**tbaep** tem como objetivo reunir as funções utilizadas internamente pela transparência Brasil no projeto Achados e Pedidos. Nesta primeira versão, temos apenas uma função que limpa os acentos dos arquivos de um diretório.

## Instalação
Para instalar, rode:
if (packageVersion("devtools") < 1.6) {
  install.packages("devtools")
}
devtools::install_github("mgaldino/tbaep")
